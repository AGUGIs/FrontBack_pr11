# 🔐 Практическое занятие №11: Управление доступом на основе ролей (RBAC)

Полнофункциональное веб-приложение с React.js фронтендом и Node.js бэкендом, реализующее JWT-аутентификацию, refresh-токены и систему управления доступом на основе ролей (RBAC).

---

## 📖 Описание проекта

Приложение реализует:

- ✅ **Регистрацию и вход** пользователей с хешированием паролей (bcrypt)
- ✅ **JWT-аутентификацию** с парой токенов (access + refresh)
- ✅ **RBAC (Role-Based Access Control)** — система ролей: Guest, User, Seller, Admin
- ✅ **Автоматическое обновление токенов** через axios interceptors
- ✅ **Хранение токенов:** access в localStorage, refresh в HttpOnly cookies
- ✅ **CRUD операции** для управления товарами и пользователями
- ✅ **Защищённые маршруты** на фронтенде и бэкенде
- ✅ **Swagger документацию** API
- ✅ **Тестовый вход** для быстрой авторизации под разными ролями
